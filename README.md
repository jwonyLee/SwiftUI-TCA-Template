## Get Started

Install dependencies
```
tuist install
```

Generate
```
tuist generate
```

Edit Structure
```
tuist edit
```

## Add Feature with Scaffold

```
tuist scaffold Feature --name Home
```

## Add Module with Scaffold

```
tuist scaffold Module --name Home
```
